# Quantica Lab — Website UI Kit

A high-fidelity recreation of the Quantica Lab marketing website (`quanticalab.ai`) based on the brand reference board.

## Surfaces covered

- Top navigation (Solutions / Products / Advisory / Insights / Contact)
- Hero with particle-wave canvas
- "What we do" feature triad
- Metrics / impact band
- Closing CTA

## Components

- `Nav.jsx` — sticky top nav with logo lockup and pill CTA
- `Hero.jsx` — split-layout hero with animated particle wave canvas
- `FeatureCard.jsx` — outlined-icon feature card with eyebrow
- `MetricsBand.jsx` — three-up impact metrics on dark
- `CTABand.jsx` — closing magenta-pink CTA section
- `Footer.jsx` — 3-column footer with logo + columns

Open `index.html` for the assembled marketing page.

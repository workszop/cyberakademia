# Quantica Lab — App UI Kit

A high-fidelity recreation of the Quantica Lab mobile dashboard — an AI dashboard surfacing model accuracy, active projects, and forecasts.

## Components
- `Header.jsx` — overview header with greeting & summary
- `MetricCard.jsx` — large pink metric card with sparkline
- `ProjectRow.jsx` — active project list row with progress
- `TabBar.jsx` — bottom tab bar (Home, Projects, Insights, Alerts, Profile)

Open `index.html` for the assembled app inside an iPhone frame.
